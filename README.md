# react-cheat-sheet

npx create-react-app my-app --no-git
rm -rf .git

Example directory structure for a Flask API and a React app:

my-app/
├── node_modules/
├── public/
│   ├── index.html
│   └── ...
├── src/
│   ├── components/
│   ├── App.js
│   ├── index.js
│   └── ...
├── package.json
├── package-lock.json
└── ...

my-api/
├── venv/
├── app/
│   ├── __init__.py
│   ├── models.py
│   ├── routes.py
│   ├── templates/
│   └── ...
├── config.py
├── requirements.txt
└── run.py
